Simply copy these files in to your Document/Playscii folder. The following features will be added to Playscii:

- Artemis default character set
- Artemis default palette
- SDA (Artemis screen dump format) export

TIPS:

- The number of available colours and characters on screen are dependant on the screen MODE command. The default is 40 x 25 characters, 8 colors. 
- SDA files can be displayed using the LOADS command
- You aren't limited to the provided characters and colors. See the commands SYMBOL and INK.
- See the main docs for more information.
